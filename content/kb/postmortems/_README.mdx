# Postmortem Template Guidance

Use this category for completed incident analyses and lessons learned.

## Expected Frontmatter Fields

- `title`
- `owner`
- `incidentId`
- `severity`
- `reviewedAt`

## Recommended Sections

## Symptoms

Summarize timeline and impact indicators.

## Diagnosis

Explain root cause and contributing factors.

## Mitigation

Capture immediate fixes and follow-up actions.

## Links

Reference related runbooks, alerts, and tickets.
