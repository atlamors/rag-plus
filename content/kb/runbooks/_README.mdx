# Runbook Template Guidance

Use this category for repeatable incident response or operational procedures.

## Expected Frontmatter Fields

- `title`
- `owner`
- `service`
- `severity`
- `reviewedAt`

## Recommended Sections

## Symptoms

Describe observable behavior and user impact.

## Diagnosis

List checks, commands, and signal interpretation.

## Mitigation

Document immediate containment and recovery steps.

## Links

Include dashboards, alerts, tickets, and related docs.
